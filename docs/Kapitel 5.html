5  Pandas

5.1 Willkommen zu Kapitel 5

Herzlich willkommen zu Kapitel 5! Nachdem wir uns in den vorherigen Kapiteln mit den Grundlagen von Python vertraut gemacht haben, werden wir uns in diesem Kapitel einem der mächtigsten Werkzeuge für die Datenanalyse widmen: der Pandas-Bibliothek.

Stellen Sie sich vor, Sie hätten eine Tabellenkalkulation wie Excel, aber mit der vollen Kraft einer Programmiersprache dahinter. Genau das ist Pandas. Es ermöglicht uns, große Datenmengen zu laden, zu bereinigen, zu transformieren und zu analysieren – alles mit wenigen Zeilen Python-Code. Ob Sie Verkaufsdaten auswerten, wissenschaftliche Messreihen analysieren oder Kundenverhalten untersuchen möchten – Pandas ist das Werkzeug der Wahl für Datenwissenschaftler und Analysten weltweit.

🎯 In diesem Kapitel werden Sie lernen
  • Was Pandas ist und wie man es installiert
  • Die grundlegenden Datenstrukturen von Pandas: Series und DataFrames
  • Wie man Daten aus verschiedenen Quellen (z.B. CSV-Dateien) lädt
  • Wie man Daten in DataFrames selektiert, filtert und bearbeitet
  • Wie man Daten aus verschiedenen Quellen zusammenführt
  • Wie man grundlegende statistische Analysen durchführt

Am Ende dieses Kapitels werden Sie in der Lage sein, reale Datensätze zu analysieren und wertvolle Einblicke aus ihnen zu gewinnen. Als Abschlussprojekt werden wir eine umfassende Datenanalyse eines echten Datensatzes durchführen. Lassen Sie uns also direkt eintauchen und die Welt der Datenanalyse entdecken!

5.2 Pandas installieren

Bevor wir mit Pandas arbeiten können, müssen wir die Bibliothek zunächst installieren. Anders als die eingebauten Python-Funktionen, die wir bisher verwendet haben, ist Pandas ein externes Paket, das von einer aktiven Community entwickelt und gepflegt wird. Diese externen Pakete erweitern die Möglichkeiten von Python enorm – und Pandas ist eines der beliebtesten davon.

Die Installation erfolgt über pip, den Paketmanager von Python. Pip ist ein Kommandozeilen-Werkzeug, das automatisch Pakete aus dem Python Package Index (PyPI) herunterlädt und installiert. Wenn Sie Python installiert haben, ist pip normalerweise bereits dabei.

Das Gute an Google Colab ist, dass die wichtigsten Datenanalyse-Bibliotheken, einschließlich Pandas, bereits vorinstalliert sind. Wenn Sie in Colab arbeiten, können Sie diesen Abschnitt überspringen und direkt mit dem Import beginnen.

💻 Installation auf einem lokalen Computer

Wenn Sie Python auf Ihrem eigenen Computer installiert haben, öffnen Sie eine Kommandozeile (Terminal auf macOS/Linux, Eingabeaufforderung oder PowerShell auf Windows) und geben Sie den folgenden Befehl ein:

pip install pandas

Pandas importieren

Nach der Installation müssen wir Pandas in unserem Python-Skript importieren, um es verwenden zu können. Hier hat sich in der Python-Community eine Konvention etabliert, die Sie in praktisch jedem Datenanalyse-Projekt wiederfinden werden: Pandas wird unter dem Alias pd importiert. Dieser kurze Alias spart Tipparbeit und macht den Code übersichtlicher.

import pandas as pd

Ab jetzt können wir alle Pandas-Funktionen über pd. aufrufen. Diese Konvention ist so verbreitet, dass Sie sie in praktisch jedem Tutorial, jeder Dokumentation und jedem professionellen Projekt finden werden.

5.3 Daten laden

Ein typischer erster Schritt in der Datenanalyse ist das Laden von Daten aus einer externen Quelle. Stellen Sie sich vor, Sie haben eine Excel-Datei mit Verkaufszahlen, eine Datenbank mit Kundeninformationen oder eine CSV-Datei mit Messwerten – wie bekommen wir diese Daten in Python, um sie analysieren zu können?

Pandas macht dies unglaublich einfach und unterstützt eine Vielzahl von Dateiformaten: CSV, Excel, JSON, SQL-Datenbanken und viele mehr. Das am häufigsten verwendete Format für tabellarische Daten ist CSV (Comma-Separated Values) – eine einfache Textdatei, in der die Werte durch Kommas getrennt sind. CSV-Dateien können von praktisch jeder Software geöffnet und erstellt werden, was sie zum universellen Austauschformat für Daten macht.

📁 Eine CSV-Datei in ein DataFrame laden

Pandas bietet die Funktion read_csv(), um Daten aus einer CSV-Datei direkt in ein DataFrame zu laden. Das funktioniert sowohl mit lokalen Dateien als auch mit URLs.

import pandas as pd

# Eine CSV-Datei von einer URL laden
url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# Eine lokale CSV-Datei laden
df = pd.read_csv("meine_daten.csv")

# Mit vollständigem Pfad
df = pd.read_csv("/pfad/zum/ordner/meine_daten.csv")

Einen ersten Blick auf die Daten werfen

Nachdem wir die Daten geladen haben, möchten wir uns meist einen ersten Überblick verschaffen. Wie viele Zeilen hat unser Datensatz? Welche Spalten gibt es? Gibt es fehlende Werte? Welche Datentypen haben die einzelnen Spalten? Diese Fragen sollten wir beantworten, bevor wir mit der eigentlichen Analyse beginnen.

Pandas bietet hierfür einige nützliche Methoden, die uns diese Fragen schnell beantworten. Mit head() und tail() können wir uns die ersten bzw. letzten Zeilen anschauen, um ein Gefühl für die Daten zu bekommen. Die Methode info() gibt uns einen technischen Überblick über die Struktur des DataFrames, während describe() automatisch statistische Kennzahlen für alle numerischen Spalten berechnet.

Die wichtigsten Methoden für diesen ersten Überblick sind head(), tail(), info() und describe(). Mit df.head() sehen Sie die ersten fünf Zeilen Ihrer Daten – ideal, um die Struktur und den Inhalt zu verstehen. Entsprechend zeigt df.tail() die letzten fünf Zeilen, was hilfreich ist, um zu prüfen, ob die Daten vollständig geladen wurden.

Besonders wertvoll ist die info()-Methode: Sie liefert eine technische Zusammenfassung mit der Anzahl der Zeilen, allen Spaltennamen, den Datentypen jeder Spalte und der Anzahl der nicht-leeren Werte. So erkennen Sie auf einen Blick, wo fehlende Werte vorliegen. Die describe()-Methode ergänzt dies durch eine statistische Zusammenfassung aller numerischen Spalten – Anzahl, Mittelwert, Standardabweichung, Minimum, Maximum und die Quartile.

# Die ersten 5 Zeilen anzeigen
print(df.head())

# Informationen über das DataFrame anzeigen
print(df.info())

# Statistische Zusammenfassung der numerischen Spalten
print(df.describe())

Das Laden und erste Erkunden von Daten ist der Ausgangspunkt für jede Datenanalyse. Mit Pandas können wir diesen Schritt schnell und effizient durchführen, um uns auf die eigentliche Analyse zu konzentrieren.

5.4 DataFrames verstehen

Um effektiv mit Pandas arbeiten zu können, müssen wir zunächst verstehen, wie Pandas Daten organisiert. Wenn Sie schon einmal mit Excel oder Google Sheets gearbeitet haben, werden Ihnen die Konzepte vertraut vorkommen. Das DataFrame ist die zentrale Datenstruktur in Pandas – man kann es sich am besten als eine Tabelle vorstellen, ähnlich wie ein Arbeitsblatt in Excel, aber mit deutlich mehr Möglichkeiten zur programmatischen Bearbeitung.

Ein DataFrame besteht aus Zeilen und Spalten. Jede Zeile repräsentiert einen Datensatz (z.B. einen Kunden, eine Messung, eine Transaktion), und jede Spalte repräsentiert ein Merkmal oder eine Variable (z.B. Name, Alter, Umsatz). Diese tabellarische Struktur ist intuitiv und entspricht der Art, wie wir Menschen Daten natürlicherweise organisieren.

🏗️ Die Anatomie eines DataFrames

Ein DataFrame besteht aus drei wesentlichen Komponenten:

  • Index: Jede Zeile hat einen eindeutigen Index (standardmäßig von 0 an aufsteigend). Dies ist vergleichbar mit der Zeilennummer in Excel.
  • Spalten: Jede Spalte hat einen Namen und enthält Daten eines bestimmten Typs (z.B. Zahlen, Text, Datum).
  • Daten: Die eigentlichen Werte, die in den Zellen der Tabelle gespeichert sind.

Eine weitere wichtige Datenstruktur: Series

Neben dem DataFrame gibt es noch eine weitere wichtige Datenstruktur, die Sie kennen sollten. Wenn Sie eine einzelne Spalte aus einem DataFrame extrahieren, erhalten Sie keine weitere Tabelle, sondern eine Series. Eine Series ist im Grunde eine eindimensionale, beschriftete Anordnung von Daten – man kann sie sich wie eine einzelne Spalte in einer Excel-Tabelle vorstellen, die auch ohne den Rest der Tabelle existieren kann.

Um eine Series zu erhalten, wählen Sie einfach eine Spalte mit eckigen Klammern aus. Der folgende Code zeigt, wie Sie die “Age”-Spalte extrahieren und überprüfen können, dass es sich tatsächlich um eine Series handelt:

import pandas as pd

url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# Eine einzelne Spalte (eine Series) auswählen
alter_spalte = df["Age"]
print(type(alter_spalte))  # <class 'pandas.core.series.Series'>

Eine Series hat wie ein DataFrame einen Index, aber nur eine einzige Spalte mit Werten. Viele der Methoden, die Sie für DataFrames kennenlernen werden – wie mean(), sum() oder value_counts() – funktionieren auch auf einer Series.

Das Verständnis der DataFrame- und Series-Strukturen ist die Grundlage für die effektive Nutzung von Pandas. Viele Operationen, die wir auf DataFrames anwenden, funktionieren auch auf Series – und umgekehrt. In den nächsten Abschnitten werden wir lernen, wie wir diese Strukturen manipulieren können, um unsere Daten zu analysieren.

5.5 Daten selektieren & filtern

Eine der häufigsten Aufgaben in der Datenanalyse ist das Auswählen von bestimmten Teilen unserer Daten. Stellen Sie sich vor, Sie haben einen Datensatz mit tausenden von Zeilen und dutzenden von Spalten – wie finden Sie genau die Informationen, die Sie brauchen? Vielleicht interessieren Sie sich nur für Kunden aus einer bestimmten Region, oder Sie möchten nur die Verkäufe des letzten Quartals analysieren.

Pandas bietet flexible und leistungsstarke Möglichkeiten, um Daten in einem DataFrame zu selektieren und zu filtern. Diese Fähigkeiten sind fundamental für die Datenanalyse, denn sie erlauben uns, aus großen Datenmengen genau die Teilmenge zu extrahieren, die für unsere Fragestellung relevant ist.

Spalten auswählen

Lassen Sie uns mit dem Einfachsten beginnen: dem Auswählen von Spalten. Die Syntax ist intuitiv und erinnert an das Arbeiten mit Dictionaries in Python. Um eine einzelne Spalte auszuwählen, verwenden wir eckige Klammern mit dem Spaltennamen: df["Spaltenname"]. Für mehrere Spalten übergeben wir eine Liste von Spaltennamen: df[["Spalte1", "Spalte2"]].

Werte zählen

Manchmal möchten wir nicht nur eine Spalte auswählen, sondern auch wissen, wie oft bestimmte Werte darin vorkommen. Die Methode value_counts() ist hierfür besonders nützlich – sie zählt automatisch, wie oft jeder eindeutige Wert in einer Spalte vorkommt. Das ist ideal für kategorische Daten wie Geschlecht, Produktkategorien oder Regionen.

import pandas as pd

url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# Wie viele Passagiere gibt es pro Klasse?
print(df["Pclass"].value_counts())

# Wie viele Passagiere sind in welchem Hafen eingestiegen?
print(df["Embarked"].value_counts())

Zeilen auswählen

Neben dem Auswählen von Spalten können wir auch gezielt bestimmte Zeilen auswählen. Hierfür bietet Pandas zwei wichtige Methoden, deren Unterschied Sie verstehen sollten: .loc und .iloc.

.loc (Label-basiert) wählt Zeilen und Spalten basierend auf ihren Labels aus – also den Indexnamen und Spaltennamen. Wenn Ihr Index aus den Standardnummern 0, 1, 2, … besteht, sind diese Nummern die Labels. Bei .loc ist der Endwert inklusiv, d.h. df.loc[0:4] gibt die Zeilen 0, 1, 2, 3 und 4 zurück.

.iloc (Integer-basiert) hingegen wählt Zeilen und Spalten basierend auf ihrer Position aus – immer von 0 an gezählt, unabhängig von den tatsächlichen Index-Labels. Bei .iloc ist der Endwert exklusiv (wie bei Python-Listen), d.h. df.iloc[0:5] gibt die Zeilen an Position 0, 1, 2, 3 und 4 zurück.

import pandas as pd

url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# .loc: Zeilen 0 bis 4, Spalten "Name" und "Age"
df.loc[0:4, ["Name", "Age"]]

# .iloc: Zeilen 0 bis 4, Spalten 2 und 4 (nach Position)
df.iloc[0:5, [2, 4]]

# Eine einzelne Zeile mit .loc
df.loc[0]

# Eine einzelne Zeile mit .iloc
df.iloc[0]

In der Praxis werden Sie .loc häufiger verwenden, wenn Sie mit Spaltennamen arbeiten, und .iloc, wenn Sie gezielt auf bestimmte Positionen zugreifen möchten – etwa die ersten 10 Zeilen oder jede zweite Zeile.

Daten filtern (Bedingte Auswahl)

Das Auswählen von Zeilen nach Position ist nützlich, aber oft möchten wir Zeilen basierend auf bestimmten Bedingungen filtern. Zum Beispiel: “Zeige mir alle Kunden, die mehr als 100 Euro ausgegeben haben” oder “Finde alle Produkte, die ausverkauft sind”. Diese Art der bedingten Auswahl ist eines der mächtigsten Werkzeuge in Pandas.

🔍 Filtern mit booleschen Bedingungen

Wir können boolesche Bedingungen verwenden, um Zeilen zu filtern, die bestimmte Kriterien erfüllen. Die Syntax mag zunächst ungewöhnlich aussehen, aber sie ist sehr mächtig.

import pandas as pd

url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# Alle Passagiere auswählen, die älter als 30 sind
aelter_als_30 = df[df["Age"] > 30]

# Alle weiblichen Passagiere in der 1. Klasse auswählen
weiblich_1_klasse = df[(df["Sex"] == "female") & (df["Pclass"] == 1)]

print(weiblich_1_klasse.head())

Wichtig: Wenn Sie mehrere Bedingungen kombinieren, müssen Sie jede Bedingung in Klammern setzen und die Operatoren & (und) oder | (oder) verwenden – nicht die Python-Schlüsselwörter and und or. Ohne die Klammern erhalten Sie einen Fehler, da Python die Operatorrangfolge anders interpretiert als erwartet.

Das Selektieren und Filtern von Daten ist ein entscheidender Schritt, um die für unsere Analyse relevanten Daten zu isolieren. Mit diesen Techniken können Sie aus großen Datensätzen genau die Informationen extrahieren, die Sie benötigen.

5.6 Daten bearbeiten

Rohdaten sind selten perfekt. In der Praxis müssen wir unsere Daten fast immer vor der Analyse bereinigen oder transformieren – fehlende Werte auffüllen, neue Spalten berechnen oder Daten in ein anderes Format umwandeln. Dieser Prozess, oft als “Data Wrangling” oder “Data Cleaning” bezeichnet, nimmt in realen Projekten oft 60-80% der Zeit in Anspruch. Pandas bietet eine Vielzahl von Werkzeugen für diese Aufgaben.

Fehlende Werte behandeln

Ein häufiges Problem in realen Datensätzen sind fehlende Werte. Vielleicht hat ein Sensor zeitweise nicht funktioniert, ein Kunde hat ein Formularfeld nicht ausgefüllt, oder Daten wurden bei der Übertragung verloren. Diese fehlenden Werte (in Pandas als NaN – “Not a Number” – dargestellt) können die Analyse verfälschen oder sogar zum Absturz unseres Codes führen, wenn wir sie nicht behandeln.

Pandas bietet zwei grundlegende Strategien für den Umgang mit fehlenden Werten: Wir können Zeilen oder Spalten mit fehlenden Werten entfernen (df.dropna()), oder wir können die fehlenden Werte mit einem sinnvollen Ersatzwert füllen (df.fillna(wert)). Welche Strategie die richtige ist, hängt vom konkreten Anwendungsfall ab – manchmal ist es besser, unvollständige Daten zu entfernen, manchmal ist es besser, sie zu ergänzen.

Neue Spalten erstellen

Oft möchten wir aus vorhandenen Daten neue Informationen ableiten. Zum Beispiel könnten wir aus dem Geburtsdatum das Alter berechnen, aus Länge und Breite die Fläche ermitteln, oder mehrere Spalten zu einer neuen kombinieren. Das Erstellen neuer Spalten ist in Pandas sehr intuitiv – wir weisen einfach einem neuen Spaltennamen einen Wert zu:

# Eine neue Spalte "Familienmitglieder" erstellen
df["Familienmitglieder"] = df["SibSp"] + df["Parch"]

Funktionen auf Spalten anwenden

Manchmal reichen einfache arithmetische Operationen nicht aus, und wir müssen komplexere Transformationen durchführen. Vielleicht möchten wir Werte in Kategorien einteilen, Text bereinigen oder eine komplexe Berechnung durchführen. Genau hier kommt die .apply()-Methode ins Spiel, die es uns ermöglicht, beliebige Python-Funktionen auf unsere Daten anzuwenden.

🔧 Die .apply()-Methode

Mit der .apply()-Methode können wir eine Funktion auf jedes Element einer Spalte anwenden. Das ist besonders nützlich, wenn wir Daten kategorisieren oder komplexe Berechnungen durchführen möchten.

# Eine Funktion, die das Alter in Kategorien einteilt
def alterskategorie(alter):
    if alter < 18:
        return "Kind"
    elif alter < 65:
        return "Erwachsener"
    else:
        return "Senior"

# Die Funktion auf die "Age"-Spalte anwenden
df["Alterskategorie"] = df["Age"].apply(alterskategorie)

Die Datenbereinigung und -transformation ist oft der zeitaufwändigste Teil der Datenanalyse, aber auch einer der wichtigsten. Nur mit sauberen, gut strukturierten Daten können wir genaue und zuverlässige Ergebnisse erzielen.

5.7 Daten sortieren

Das Sortieren von Daten ist eine häufige Operation, um Muster zu erkennen oder die größten bzw. kleinsten Werte zu finden. Wer sind unsere Top-10-Kunden? Welche Produkte haben die schlechtesten Bewertungen? Welche Mitarbeiter haben die meisten Überstunden? Sortierung hilft uns, solche Fragen schnell zu beantworten und die interessantesten Datenpunkte zu identifizieren.

🔢 Die sort_values()-Methode

Mit sort_values() können wir ein DataFrame nach einer oder mehreren Spalten sortieren. Standardmäßig wird aufsteigend sortiert, aber das lässt sich leicht ändern.

import pandas as pd

url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# Nach Alter sortieren (aufsteigend)
df_sortiert = df.sort_values("Age")

# Nach Alter sortieren (absteigend)
df_sortiert = df.sort_values("Age", ascending=False)

# Nach mehreren Spalten sortieren
df_sortiert = df.sort_values(["Pclass", "Age"], ascending=[True, False])

print(df_sortiert.head())

Im letzten Beispiel sehen Sie, wie wir nach mehreren Spalten sortieren können – zuerst nach Klasse (aufsteigend), dann innerhalb jeder Klasse nach Alter (absteigend). Diese Technik ist besonders nützlich, wenn Sie Ranglisten oder Berichte erstellen möchten.

5.8 Daten zusammenfügen

In der Praxis liegen unsere Daten oft nicht in einer einzelnen Tabelle vor. Stellen Sie sich vor, Sie haben eine Tabelle mit Kundeninformationen (Name, Adresse, Kundennummer) und eine separate Tabelle mit Bestellungen (Bestellnummer, Kundennummer, Produkt, Preis). Um herauszufinden, welcher Kunde was bestellt hat, müssen wir diese Tabellen zusammenführen – eine Operation, die in der Datenbankwelt als “JOIN” bekannt ist.

Pandas bietet verschiedene Möglichkeiten, um DataFrames zu kombinieren. Die zwei wichtigsten sind pd.concat() und pd.merge(). Mit concat() fügen wir DataFrames entlang einer Achse zusammen – wie das Stapeln von Tabellen übereinander oder nebeneinander. Mit merge() kombinieren wir DataFrames basierend auf gemeinsamen Spalten, ähnlich wie ein SQL-JOIN, bei dem Zeilen aus verschiedenen Tabellen anhand eines gemeinsamen Schlüssels verknüpft werden.

Lassen Sie uns die merge()-Funktion an einem praktischen Beispiel betrachten, um zu verstehen, wie sie funktioniert.

🔄 Beispiel: merge()

Stellen Sie sich vor, wir haben zwei DataFrames: eines mit Bestelldaten und eines mit Kundendaten. Wir können sie über die gemeinsame Spalte KundenID zusammenführen.

import pandas as pd

bestellungen = pd.DataFrame({
    "BestellID": [1, 2, 3, 4],
    "KundenID": [101, 102, 101, 103],
    "Produkt": ["A", "B", "C", "A"]
})

kunden = pd.DataFrame({
    "KundenID": [101, 102, 103],
    "Name": ["Anna", "Max", "Julia"]
})

# Die beiden DataFrames zusammenführen
merged_df = pd.merge(bestellungen, kunden, on="KundenID")

print(merged_df)

Das Ergebnis ist ein neues DataFrame, in dem jede Bestellung mit den entsprechenden Kundeninformationen verknüpft ist. Das Zusammenfügen von Daten aus verschiedenen Quellen ist eine wesentliche Fähigkeit, um ein vollständiges Bild Ihrer Daten zu erhalten.

5.9 Grundlegende Statistiken mit Pandas

Der wahre Wert der Datenanalyse liegt darin, aus Zahlen Erkenntnisse zu gewinnen. Rohdaten allein sagen uns wenig – erst wenn wir sie zusammenfassen, vergleichen und Muster erkennen, werden sie wertvoll. Pandas macht es sehr einfach, grundlegende statistische Kennzahlen für unsere Daten zu berechnen. Diese Kennzahlen geben uns einen schnellen Überblick über die Verteilung und die zentralen Tendenzen unserer Daten.

Die wichtigsten statistischen Methoden in Pandas sind:

  • .mean(): Berechnet den Mittelwert (Durchschnitt)
  • .median(): Berechnet den Median (mittlerer Wert)
  • .std(): Berechnet die Standardabweichung (Streuung der Werte)
  • .min() und .max(): Finden den kleinsten und größten Wert
  • .sum(): Berechnet die Summe aller Werte
  • .count(): Zählt die Anzahl der nicht-leeren Werte

Jede dieser Methoden kann auf eine einzelne Spalte oder das gesamte DataFrame angewendet werden. Diese Methoden sind nützlich für einen schnellen Überblick, aber oft möchten wir Statistiken für verschiedene Gruppen berechnen – zum Beispiel den Durchschnittsumsatz pro Region oder das Durchschnittsalter pro Geschlecht.

groupby(): Gruppieren und Aggregieren

Die groupby()-Funktion ist eines der mächtigsten Werkzeuge in Pandas. Sie ermöglicht es uns, Daten nach Kategorien zu gruppieren und dann Berechnungen auf jede Gruppe anzuwenden – ähnlich wie eine Pivot-Tabelle in Excel. Mit groupby() können Sie Fragen beantworten wie “Wie hoch ist der durchschnittliche Bestellwert pro Kundengruppe?” oder “Wie viele Verkäufe gab es pro Monat?”

🔄 Die mächtige groupby()-Funktion 
import pandas as pd

url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# Das Durchschnittsalter nach Geschlecht gruppieren
durchschnittsalter_pro_geschlecht = df.groupby("Sex")["Age"].mean()

print(durchschnittsalter_pro_geschlecht)

Statistische Analysen sind der Kern der Datenanalyse. Mit den grundlegenden Methoden und der groupby()-Funktion haben Sie bereits ein mächtiges Werkzeugset, um aus Ihren Daten wertvolle Erkenntnisse zu gewinnen.

Lösung zur Übung: Datenanalyse

import pandas as pd

# 1. Daten laden
url = ("https://raw.githubusercontent.com/"
       "datasciencedojo/datasets/master/titanic.csv")
df = pd.read_csv(url)

# 2. Fehlende Alterswerte mit dem Median füllen
median_age = df["Age"].median()
df["Age"] = df["Age"].fillna(median_age)

# 3. Analyse
# Frage 1: Allgemeine Überlebensrate
allgemeine_ueberlebensrate = df["Survived"].mean()
print(f"Allgemeine Überlebensrate: {allgemeine_ueberlebensrate:.2%}\n")

# Frage 2: Überlebensrate nach Geschlecht
ueberlebensrate_geschlecht = df.groupby("Sex")["Survived"].mean()
print("Überlebensrate nach Geschlecht:")
print(ueberlebensrate_geschlecht)
print("\n")

# Frage 3: Überlebensrate nach Klasse
ueberlebensrate_klasse = df.groupby("Pclass")["Survived"].mean()
print("Überlebensrate nach Klasse:")
print(ueberlebensrate_klasse)
print("\n")

# Frage 4: Zusammenhang zwischen Alter und Überleben
# Erstellen von Altersgruppen
altersgruppen = pd.cut(df["Age"], bins=[0, 18, 35, 60, 100], labels=["Kind", "Junger Erwachsener", "Erwachsener", "Senior"])
ueberlebensrate_alter = df.groupby(altersgruppen)["Survived"].mean()
print("Überlebensrate nach Altersgruppe:")
print(ueberlebensrate_alter)

Interpretation der Ergebnisse

Was sagen uns diese Zahlen? Die Analyse liefert aufschlussreiche Erkenntnisse über die Überlebensraten auf der Titanic.

Die allgemeine Überlebensrate lag bei etwa 38% – nur etwa jeder dritte Passagier überlebte das Unglück. Das zeigt bereits, wie verheerend die Katastrophe war.

Der Faktor Geschlecht war entscheidend: Frauen hatten eine deutlich höhere Überlebensrate von etwa 74%, während nur etwa 19% der Männer überlebten. Dies spiegelt die “Frauen und Kinder zuerst”-Politik bei der Evakuierung wider, die offensichtlich konsequent umgesetzt wurde.

Der soziale Status spielte ebenfalls eine erhebliche Rolle: Passagiere der 1. Klasse hatten die höchste Überlebensrate mit etwa 63%, gefolgt von der 2. Klasse mit etwa 47% und der 3. Klasse mit nur etwa 24%. Die Kabinen der 1. Klasse lagen näher an den Rettungsbooten, und wohlhabendere Passagiere hatten vermutlich besseren Zugang zu Informationen und Rettungsmitteln.

Das Alter zeigt ebenfalls interessante Muster: Kinder hatten eine höhere Überlebensrate als die anderen Altersgruppen, was die Beobachtung unterstützt, dass Kinder bei der Evakuierung bevorzugt wurden.

7.5 Übung 3: Algorithmus-Challenge

Ziel: Berechnen Sie die Summe aller Zahlen in einem bestimmten Bereich, die durch 7 teilbar sind.

Anforderungen

  1. Zahlenbereich definieren: Alle Zahlen von 18 bis einschließlich 534 betrachten

  3. Summe berechnen: Alle teilbaren Zahlen aufsummieren

  4. Ergebnis ausgeben: Die berechnete Summe mit print() anzeigen

  2. Erstellen Sie eine for-Schleife, die alle Zahlen von 18 bis 534 durchläuft.

  4. Wenn ja, addieren Sie die Zahl zur Summe.
  5. Geben Sie nach der Schleife das Ergebnis aus.

    # Tipp: zahl % 7 == 0 bedeutet "kein Rest bei Division durch 7"

  • Modulo-Operator %: Gibt den Rest einer Division zurück

7.7 Übung 5: Sentiment-Analyse

Das finale Projekt ist das anspruchsvollste Aufgabenset und bringt alles zusammen, was Sie in diesem Kurs gelernt haben. Sie führen eine vereinfachte Sentiment-Analyse durch – eine Technik, die in der Praxis häufig verwendet wird, um die Stimmung in Texten automatisch zu analysieren.

Ziel: Führen Sie eine vollständige Sentiment-Analyse eines Nachrichtendatensatzes durch, indem Sie positive, negative und neutrale Nachrichten identifizieren und den Gesamtstimmungswert berechnen.

Anforderungen

  1. Datensatz analysieren: Die Anzahl der Nachrichten im Datensatz ermitteln

  2. Positive Nachrichten:

    • Zählen, wie viele Nachrichten mindestens ein positives Wort enthalten
    • Den Anteil positiver Nachrichten berechnen

  3. Negative Nachrichten:

    • Zählen, wie viele Nachrichten mindestens ein negatives Wort enthalten
    • Den Anteil negativer Nachrichten berechnen

  4. Stimmungswert berechnen: Anteil positiv minus Anteil negativ

  5. Gesamtstimmung bewerten: Mit if-Anweisung die Stimmung als positiv, negativ oder neutral einordnen

  6. Neutrale Nachrichten: Den Anteil neutraler Nachrichten berechnen

Anleitung

  1. Verwenden Sie len(), um die Anzahl der Nachrichten zu ermitteln.
  2. Erstellen Sie eine verschachtelte Schleife: äußere Schleife für Nachrichten, innere für Wörter.
  3. Prüfen Sie mit if wort in nachricht, ob ein Schlüsselwort enthalten ist.
  4. Zählen Sie positive und negative Nachrichten separat.
  5. Berechnen Sie Anteile durch Division: anzahl / gesamtanzahl.
  6. Der Stimmungswert ist: anteil_positiv - anteil_negativ.
  7. Neutrale Nachrichten sind weder positiv noch negativ kategorisiert.

Code-Struktur

# Gegebene Daten (werden im Notebook bereitgestellt)

# positive_wörter = [...]  # Liste mit 14 positiven Begriffen

print(f"Anzahl Nachrichten: {anzahl_nachrichten}")

# Aufgabe 17-18: Positive Nachrichten analysieren

for nachricht in nachrichten:
    ist_positiv = False
    for wort in positive_wörter:

            break  # Schleife beenden, sobald ein Wort gefunden wurde
    if ist_positiv:

print(f"Positive Nachrichten: {positive_count} ({anteil_positiv:.2%})")

negative_count = 0
for nachricht in nachrichten:
    ist_negativ = False
    for wort in negative_wörter:
        if pass:  # Ersetzen: wort in nachricht
            ist_negativ = True
            break
    if ist_negativ:
        negative_count = pass  # Ersetzen: negative_count + 1

print(f"Negative Nachrichten: {negative_count} ({anteil_negativ:.2%})")

print(f"Stimmungswert: {stimmungswert:.2f}")

    print("Die Gesamtstimmung ist POSITIV")
elif pass:  # Ersetzen: stimmungswert < 0
    print("Die Gesamtstimmung ist NEGATIV")
else:
    print("Die Gesamtstimmung ist NEUTRAL")

# Aufgabe 23: Anteil neutraler Nachrichten
# Hinweis: Neutral = weder positiv noch negativ kategorisiert
# Dies erfordert eine kombinierte Prüfung oder Berechnung
pass  # Ersetzen Sie dies durch Ihren Code

Hinweise

  • Verschachtelte Schleifen: Eine äußere Schleife für Nachrichten, eine innere für Wörter
  • in-Operator: "gut" in "Das ist gut" ergibt True
  • break: Beendet die innere Schleife vorzeitig, wenn ein Wort gefunden wurde
  • Anteil berechnen: anzahl / gesamtanzahl (ergibt eine Dezimalzahl zwischen 0 und 1)
  • Formatierung: {wert:.2%} zeigt den Wert als Prozent mit 2 Dezimalstellen
  • Stimmungswert: Positiv wenn > 0, negativ wenn < 0, neutral wenn = 0

Der Algorithmus im Detail

Der grundlegende Ansatz für die Wortsuche:

  1. Zähler auf 0 setzen
  2. Jede Nachricht durchlaufen
  3. Für jede Nachricht: Hilfsvariable ist_positiv auf False setzen
  4. Alle Schlüsselwörter prüfen
  5. Wenn ein Wort gefunden wird: ist_positiv = True und break
  6. Nach der inneren Schleife: Wenn ist_positiv, Zähler erhöhen
  7. Am Ende: Anteil = Zähler / Gesamtanzahl

7.8 Erfolgsstrategien

Eine gute Vorbereitung ist der halbe Erfolg. Bevor Sie mit dem Test beginnen, möchte ich Ihnen einige bewährte Strategien mit auf den Weg geben, die Ihnen helfen werden, die Aufgaben erfolgreich zu meistern.

  • Listenzugriff: Index beginnt bei 0, nicht bei 1
  • Modulo-Operator: % gibt den Rest einer Division zurück
  • Verschachtelte Schleifen: Bei der Sentiment-Analyse benötigen Sie eine Schleife in einer Schleife

Mit diesem Wissen sind Sie bestens gerüstet, um den Test erfolgreich zu meistern. Nehmen Sie sich die Zeit, die Sie brauchen, und arbeiten Sie die Aufgaben sorgfältig durch. Es gibt keinen Zeitdruck – Qualität geht vor Schnelligkeit.

7.9 Nach dem Test

Unabhängig davon, ob Sie den Test beim ersten Versuch bestehen oder nicht – Sie haben bereits einen großen Schritt in Ihrer Programmierreise gemacht. Das Wichtigste ist, dass Sie dranbleiben und weiter lernen.

Bei erfolgreichem Abschluss können Sie stolz auf sich sein und Ihren Erfolg in der Community feiern. Nutzen Sie das Momentum und entwickeln Sie eigene kleine Projekte, um Ihre neuen Fähigkeiten anzuwenden und zu festigen. Je mehr Sie programmieren, desto natürlicher wird es sich anfühlen.

Ihre Weiterbildungsmöglichkeit: Business Data Analyst (IHK)

Wenn Sie Ihre Python-Kenntnisse vertiefen und professionell im Bereich Datenanalyse arbeiten möchten, ist der Business Data Analyst (IHK) Kurs der ideale nächste Schritt. Dieser 8-wöchige Blended-Learning-Kurs baut direkt auf den Grundlagen auf, die Sie hier erworben haben.

In den Wochen 1-2 lernen Sie die Rolle des Data Analysts kennen und vertiefen Ihre statistischen Grundlagen. Sie beschäftigen sich mit Kennzahlen, Stichproben, dem Unterschied zwischen Korrelation und Kausalität sowie professioneller Datenvisualisierung.

Die Wochen 3-4 widmen sich der Python-Vertiefung und der Datenanalyse mit Pandas. Sie arbeiten intensiv mit DataFrames, lernen Datenmanipulation, Filterung und statistische Auswertung.

In den Wochen 5-6 erstellen Sie professionelle Diagramme mit Matplotlib – von Linien- und Streudiagrammen bis hin zu Balken- und Kuchendiagrammen. Sie lernen auch, Fehler in Visualisierungen zu erkennen und den richtigen Diagrammtyp für Ihre Daten zu wählen.

Die Wochen 7-8 führen Sie in Tableau ein, ein mächtiges Tool für interaktive Dashboards. Sie lernen sowohl Grundlagen als auch fortgeschrittene Techniken und erstellen anspruchsvolle Dashboards für datenbasierte Erkenntnisse.

Als Bonus erhalten Sie einen Projektleitfaden zur erfolgreichen Umsetzung von Datenanalyse-Projekten in der Praxis.

Das Format umfasst 78 Unterrichtsstunden (20 UE Live-Online + 58 UE Selbstlernstudium) und schließt mit einem Abschlussprojekt und IHK-Zertifikat ab.
